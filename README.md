# Tutorial: Mechanizing the Nano-P4 Specification with P4-SpecTec

A guided tutorial for learning P4-SpecTec by mechanizing the semantics of Nano-P4,
an educational dialect of P4. Covers the basics of P4-SpecTec, static semantics,
dynamic semantics, and prose generation.

## Serving locally

```sh
mdbook serve --open
```
