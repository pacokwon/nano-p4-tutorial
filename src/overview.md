# Overview

This guide will walk you through the process of spec mechanization in P4-SpecTec.
We will take the basics of P4-SpecTec to describe the semantics of Nano-P4,
an educational dialect of P4.

At the end of the tutorial, you will have a full mechanized specification, a reference
type checker and interpreter, and a natural language documentation for Nano-P4.
Most importantly, you will gain hands-on experience writing a mechanized specification
using P4-SpecTec, a skill that can be transferred to any other project using the toolchain,
including the full mechanized specification for the P4 language.
